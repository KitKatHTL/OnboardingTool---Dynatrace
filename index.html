Deployment & GitHub Pages

  1. Erstelle ein neues GitHub-Repository und push den Inhalt dieser HTML-Datei.
  2. Aktiviere GitHub Pages in den Repository-Einstellungen.
  3. Öffne die URL, um die Seite anzuzeigen.
