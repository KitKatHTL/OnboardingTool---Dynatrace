Projektbeschreibung

Erfahren Sie mehr über die technischen Details und Funktionen unseres Onboarding Tools

Ziel

Das Tool unterstützt den Bewerbungs- und Onboarding-Prozess mit Automatisierungen, Checklisten und zentralen Übersichten, um New Starter effizient einzugliedern.

Funktionen

Nutzerverwaltung

Bewerbungsunterlagen, Statusverwaltung, Archivierung

Checklisten

Standard und individuelle Anpassung

Automatisierungen

Jira, Miro, Slack, OpenAPI Integration

Dashboard

Übersicht über alle Prozesse und Fortschritte

Technische Basis

Backend

Java (Spring Boot)

Frontend

React mit Material Design

Datenbank

PostgreSQL

Integrationen

Jira API, Miro API, Slack API, OpenAPI
